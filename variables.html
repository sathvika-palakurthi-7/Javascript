JavaScript Variables Demonstration

Learn about var, let, and const, and how JavaScript variables work

1. Variable Declaration (var, let, const)

JavaScript has three ways to declare variables:

var age = 25; // Function-scoped, can be redeclared
let name = "John"; // Block-scoped, cannot be redeclared
const city = "NewYork"; // Block-scoped, cannot be reassigned
